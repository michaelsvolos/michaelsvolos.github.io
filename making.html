Simulating the Reverberation of Cities

For my honors capstone project, I used the image source method to simulate the reverberant sound of city streets. Impulse responses were created in MATLAB for use in convolution reverb plugins. Paper Demo

Classifying Musical Scores by Instrument

This was a project for CS 221 - Artificial Intelligence: Principles and Techniques. We created an algorithm that takes digital representations of individual instrument parts from Classical-era pieces and classifies them by instrument. Paper

Switchboard

Switchboard is a new musical instrument created for Music 250A - Physical Interaction Design. It was built using Faust running on an Arduino Teensy 3.6. Video demo

Adtune

Adtune is a computer music-mediated reminder of just how many ads are out there. It's a Google Chrome extensions that, when activated, counts the number of ads on the current website and synthesizes and plays a song. On a site with no ads, the music is relaxing, but this quickly changes once the number of ads goes up. Adtune is a project for Treehacks 2019. Github
